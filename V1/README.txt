# V1: Matrix Multiplier ASIC Baseline

## Overview
The directory contains the initial V1 release of the 8x8 systolic array matrix processor. 

## Project Structure
- **/RTL**: Verilog source files for the matrix multiplier core and control logic.
- **/Cocotb**: CoCoTB-based testbenches and register-level verification scripts.
- **/PD**: Physical design constraints, synthesis scripts, and reports.
